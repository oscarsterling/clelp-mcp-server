# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in the Clelp MCP Server, please report it responsibly.

**Do NOT open a public issue for security vulnerabilities.**

Instead, email security concerns to: jason@clelp.ai

We'll acknowledge receipt within 48 hours and aim to provide a fix or mitigation plan within 7 days.

## Scope

This policy covers the `clelp-mcp-server` npm package and its interaction with the Clelp API.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.1.x   | Yes       |
| < 1.1   | No        |
